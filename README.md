#  Inventory Management System (MySQL)

A professional Inventory Management System built entirely in **MySQL** to simulate how real businesses manage inventory, suppliers, purchases, sales, stock movement, and business reporting.

This project demonstrates advanced SQL concepts including **Stored Procedures, Triggers, Views, Transactions, Indexing, Query Optimization, and Business Intelligence Reports**.

---

#  Project Overview

This project manages the complete inventory lifecycle:

- Supplier Management
- Product Management
- Purchase Tracking
- Sales Tracking
- Inventory Updates
- Automated Stock Management
- Audit Logging
- Executive Business Reporting

The database is designed to mimic how inventory systems work in real companies.
